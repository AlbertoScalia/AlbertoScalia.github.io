# Alberto Scalia – Sito Personale

Questo repository contiene il codice sorgente del mio portfolio personale, un progetto sviluppato per unire estetica e performance tecnica. Il lavoro è nato come progetto pratico all'interno del corso di HTML e CSS di start2impact.

## Caratteristiche Tecniche

Il progetto è stato sviluppato seguendo un approccio "clean code", ottimizzando ogni riga per la velocità e la leggibilità:

* **HTML5 & SEO**: Struttura semantica avanzata per massimizzare la visibilità sui motori di ricerca.
* **SCSS**: Architettura modulare dei fogli di stile con gestione avanzata di variabili per la palette Neon Green & Deep Dark.
* **Bootstrap 5**: Framework utilizzato per garantire una responsività fluida su ogni dispositivo (Mobile First).
* **EmailJS**: Gestione dinamica del modulo contatti, permettendo l'invio di email direttamente dal client senza un backend dedicato.
* **JSON-LD**: Integrazione di schema per un'indicizzazione semantica ottimale del profilo professionale.

## Concept Visivo: Sintesi Visiva

L'interfaccia riflette la mia filosofia di design, basata sulla riduzione del superfluo e sulla forza del contrasto:

1. **Tipografia**: Utilizzo di font sans-serif moderni con pesi tipografici estremi per creare una chiara gerarchia visiva.
2. **Palette Colori**: Contrasto netto tra il nero assoluto del fondo e accenti "Neon Green" per guidare l'utente verso le Call-to-Action.
3. **User Experience**: Navigazione essenziale e priva di distrazioni, focalizzata esclusivamente sulla sezione Works.
